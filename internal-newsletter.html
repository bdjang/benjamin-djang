HTML Email: Internal Newsletter

Launched: Ongoing since September 2012

At Democracy Prep, I manage a newsletter sent to our internal staff. Through this project, I learned about designing and coding HTML emails and using email data to improve open rates. Below is an overview of how the newsletter has changed and developed.

Template #1

BACKGROUND:

I first started out sending internal email newsletters by using a WYSIWYG editor and a pre-made email template from Verticalresponse.com. I completed the HTML/CSS course on Codecademy to learn more about coding and customizing HTML emails.

Redesign #1

BACKGROUND:

I designed and created an email template that reflected the needs of Democracy Prep's internal newsletter.

Redesign #2

BACKGROUND:

I became more interested in designing and creating HTML emails. I attended the Responsive Email Newsletter Design workshop and started reading books and blogs about HTML emails. During the 2014 summer, I redesigned the newsletter to be more user friendly and mobile responsive.
