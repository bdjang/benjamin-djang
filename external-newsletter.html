HTML Email: External Newsletter

Launched: Ongoing since September 2012

At Democracy Prep, I manage a monthly external newsletter sent to our external audience. Below is an overview of the changes and improvements I made to the newsletter.

Template #1

BACKGROUND:

I first started out sending external emails by using a WYSIWYG editor and a pre-made email template from Verticalresponse.com. I completed the HTML/CSS course on Codecademy to learn more about coding and customizing.

Redesign #1

BACKGROUND:

In December 2012, I designed and coded an email template to improve my organization's communication with our external audience.

Redesign #2

BACKGROUND:

I became more interested in designing and creating HTML emails. I attended the Responsive Email Newsletter Design workshop and started reading books and blogs about HTML emails. During the 2014 summer, I redesigned the newsletter to be more user friendly and mobile responsive.

Redesign #3

BACKGROUND:

Through using Litmus, I started to code the external newsletter template to render correctly in our external audience's most used email clients and devices.
